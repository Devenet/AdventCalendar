Advent Calendar /about/

What is this?

Advent Calendar is a very light web application to show a picture and its legend per day before Christmas, or any other event.

It’s written in PHP, use last web technologies—and proud to not support obsolete versions of IE.

Want to contribute?

Source code is developed by Nicolas Devenet and hosted on GitHub. Feel free to fork it and to improve the application!
